Description


Implement a Queue by linked list. Support the following basic methods:
```
enqueue(item). Put a new item in the queue.
dequeue(). Move the first item out of the queue, return it.
```

Example
```
Example 1:

Input:
enqueue(1)
enqueue(2)
enqueue(3)
dequeue() // return 1
enqueue(4)
dequeue() // return 2

Example 2:

Input:
enqueue(10)
dequeue()// return 10
```
