Given a mountain sequence of n integers which increase firstly and then decrease, find the mountain top.
Arrays are strictly incremented, strictly decreasing

Example
```
Example 1:

Input: nums = [1, 2, 4, 8, 6, 3] 
Output: 8

Example 2:

Input: nums = [10, 9, 8, 7], 
Output: 10
```
