Find the last position of a target number in a sorted array. Return -1 if target does not exist.

Example
```
Example 1:

Input: nums = [1,2,2,4,5,5], target = 2
Output: 2

Example 2:

Input: nums = [1,2,2,4,5,5], target = 6
Output: -1
```
