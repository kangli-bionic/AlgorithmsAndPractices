Given a list of numbers with duplicate number in it. Find all unique permutations.

Example
```
Example 1:

Input: [1,1]
Output:
[
  [1,1]
]

Example 2:

Input: [1,2,2]
Output:
[
  [1,2,2],
  [2,1,2],
  [2,2,1]
]
```
Challenge

Using recursion to do it is acceptable. If you can do it without recursion, that would be great!
